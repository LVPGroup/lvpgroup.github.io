Formal Verification of a GlobalPlatform Specification compliant MiTEE

Theories

Theory Contents
SecurityModel This is the security model of MiTEE, including several security objectives while this paper focus on the isolation. Information-flow properties are defined here.
Function_Layer This layer includes the definitions of basic data structures and execution model, event specification, the instantiation proof, and security proofs of events.
Design_Layer This layer includes the model of design layer, refinement proof between function layer and design layer, the security proof on new variables in design layer, and the final security proof of MiTEE.
