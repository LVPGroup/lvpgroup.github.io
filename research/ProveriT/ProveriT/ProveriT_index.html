ProveriT File Content Introduction

SPD Layer includes the data model which formalizes security assets in GPTEE PP, the threat layer state machine TSM, and the formal definition of security threats in GPTEE PP.

SO Layer is the formalization of security objectives in GPTEE PP, including the objective layer state machine OSM, the formal definition of security objectives, and the rationales between security threats and security objectives.

HCL and SFR Layer is the formalization of security functional requirements in GPTEE PP, including the requirement layer state machine RSM, the data structures for the high composition layer (HCL), definitions of HCL, rationales between SOs and HCL SFRs, definitions of SFRs, rationales between HCL SFRs and SFRs.

Theories

  • SPD Layer: Security Problem Definitions
  • SO Layer: Security Objectives
  • HCL and SFR Layer
